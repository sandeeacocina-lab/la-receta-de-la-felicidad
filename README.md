# La Receta de la Felicidad

Sitio estático bilingüe de recetas de Sandra Mangas, optimizado para GitHub Pages.

## Publicación

GitHub Actions reconstruye automáticamente el paquete de `.site-parts/`, verifica su integridad y publica todos los archivos en GitHub Pages.

El dominio definitivo será `larecetadelafelicidad.com`, pero no está conectado todavía. Hasta completar las pruebas se usa la dirección provisional de GitHub Pages.

Los detalles de la migración y las comprobaciones realizadas se encuentran dentro del sitio, en `MIGRACION.md`.
